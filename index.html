Genlink — Intergenerational Platform

Genlink is a full-stack web application designed to bridge the generational gap by connecting youth and elderly through shared stories, meaningful friendships, and community engagement. The platform encourages intergenerational bonding by providing a safe, welcoming space where users of all ages can interact.

Accessibility-first design: The interface features adjustable text sizes for readability, a dark mode toggle for comfortable browsing, and multilingual support powered by Google Neural Machine Translation — ensuring the platform is usable for both seniors and digital-native youth.

Secure authentication system: Robust user authentication with comprehensive input validation, account lockout policies to prevent brute-force attacks, and time-sensitive password reset tokens delivered via Gmail SMTP with TLS encryption to safeguard user trust and data privacy.

Backend & deployment: Built with Python Flask and Jinja templating on the frontend, with MySQL hosted on Railway for reliable, scalable data storage. The entire application is deployed on Render.com for seamless, always-on access.

Hospital Patient Flow & Efficiency Dashboard

Analysed 5 real government datasets from data.gov.sg, OECD, and SingStat to build an interactive Power BI dashboard exploring hospital patient flow patterns across Singapore.

Created custom DAX measures for dynamic year filtering, KPI cards tracking bed capacity growth (+4.6%), average wait times, and demographic breakdowns. Key findings revealed that 34.3% of daily patients arrive during the 12–6 PM peak window, with the 65+ age group accounting for 62.77% of admissions — insights that directly inform staffing and capacity planning decisions.
